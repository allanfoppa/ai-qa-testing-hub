# AI QA Testing Hub

A centralized tool-agnostic for multi-project quality assurance.

## 🚀 Quick Start
1. `npm install`
2. Update `config/apps-registry.json` with your local service URLs.
3. Run tests for a specific project:
   `npm test -- suites/backend-freight-api`
